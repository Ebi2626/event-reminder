Pozwala definiować wydarzenia cykliczne i wysyłać przypomnienia email.

== Description ==

Wtyczka umożliwia:
* Tworzenie wydarzeń cyklicznych (MM-DD)
* Wysyłanie emaili: 30, 14, 7, 3, 1 dzień przed
* Panel admin dla wydarzeń
* WP Cron do wysyłki

== Installation ==

1. Wgraj `event-reminder` do `/wp-content/plugins/`
2. Aktywuj wtyczkę
3. Dodaj wydarzenie: Events → Add New
4. Dodaj emaile w metaboksie
